Museu Digital é uma exposição virtual criada para oferecer uma experiência acessível, interativa e moderna de apreciação artística. O objetivo do projeto é democratizar o acesso à arte e estimular a curiosidade cultural através de uma plataforma intuitiva, responsiva e envolvente.

Este projeto é fictício e foi desenvolvido para fins educacionais e de portfólio, utilizando HTML, CSS e JavaScript puros.
